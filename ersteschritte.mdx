## Folgende Schritte zum durchstarten mit ZAUNBOT

### Schritt 1: Vorraussetzungen erfüllen

Um zu garantieren, dass alles gut auf deinem Server funktioniert, achte darauf, folgende **Voraussetzungen** zu erfüllen:

- Du brauchst einen Discord-Server, auf dem du über **Administrator-Berechtigungen** verfügst.
- das wars...

### Schritt 2: Bot invite

Lade unseren **Discord-Bot** per **Invite-Link** auf deinen Server ein. Wichtig ist, dass die Rolle, die der Bot automatisch auf deinem Server bekommt, in der Rollenliste nach ganz oben geschoben wird, um Probleme mit Berechtigungen zu vermeiden.

<Warning>
  Alle Funktionen des Bots sind für deinen Server noch deaktiviert, sofern noch kein Kauf getätigt wurde. Features werden bei Kauf entsprechend freigeschaltet.
</Warning>

### Schritt 3: Features erwerben

Als nächstes müssen Features für deinen Server auf unserem Discord-Server erworben werden. Sobald der Kauf **abgeschlossen** ist, wird das Feature für deinen Server **freigeschaltet**. Über entsprechende Commands kann das Feature entsprechend verwendet werden.

> Es kann sein, dass das Feature als **Lizenz-Key** ausgestellt wird. Der Key muss per Command auf deinem Server eingelöst werden, um es zu aktivieren.
