## Folgende Schritte zum durchstarten mit ZAUNBOT

### Schritt 1: Vorraussetzungen erfüllen

Um zu garantieren, dass alles gut auf deinem Server funktioniert, achte darauf folgende Vorraussetzungen zu erfüllen:

1. du brauchts einen Discord-Server, auf dem du über Administrator-Berechtigungen verfügst.

### Schritt 2: Bot invite

Lade unseren Discord-Bot per Invite-Link auf deinen Server ein. Wichtig ist, das die Rolle, welcher der Bot automatisch auf deinem Server bekommt, in der Rollenliste nach ganz oben geschiben wird, um Probleme mit Berechtigungen zu vermeiden.

<Warning>
  Alle Funktionen des Bots sind für deinen Server noch deaktiviert, sofern noch kein Kauf getätigt wurde. Features werden bei Kauf entsprechend freigeschaltet.
</Warning>
