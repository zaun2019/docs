### Wie funktioniert es?

Unser Lizenz-System ermöglicht es Kunden einzelne Features zu erwerben, ohne direkt alles kaufen zu müssen. Somit kann sich der User beim Kauf auf das beschräänken was er wirkich braucht. Es ist jeder Zeit möglich nachträgich Features zu erwerben, um System zu erweitern.

Beim Erwerb wird das Feature nach Kaufaschluss für den entsprechenden Käufer freigeschaltet. Alternativ kann ein Lizenz-Key generiert werden, welchen der Käufer eigenständig per Command ausführen kann, um das Feature zu aktivieren.
